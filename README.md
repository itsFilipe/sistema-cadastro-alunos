Sistema simples de cadastro de alunos desenvolvido em C, onde é possível inserir, atualizar e deletar cadastro de alunos;
O cadastro possui Número da matrícula, que é gerado automaticamente, nome, idade, curso, nota média.

Existem 8 opções a serem escolhida pelo usuário, que são organizadas por um switch;

1 - Adicionar aluno
2 - Buscar pela matricula
3 - Buscar pelo primeiro nome
4 - Buscar por curso
5 - Contagem dos alunos
6 - Excluir aluno
7 - Atualizar aluno
8 - Sair
